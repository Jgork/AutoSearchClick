1. Делаем скрины кнопок, по которым надо нажимать и закидываем их в любую папку.
2. Выбираем режим клика (Сначала режим, потом выбираем папку. Именно в таком порядке. При смене режима, папку выбираем заново).
3. Выбираем папку со скринами (из 1 шага).
4. Жмете F7 и отдыхаете.

Теперь про режимы.
"Мультиклик" и "По порядку" можно (рекоммендуется) использовать одновременно.

*Мульти - программа будет тыкать пока не нажмете стоп (без галочки нажмет 1 раз по первому месту, которое найдёт).
*По порядку - будет тыкать в том порядке, в котором лежат скрины в папке (для удобства рекоммендую переименовать скрины в числа (1,2,3...), в том порядке, в котором хотите видеть клики).
